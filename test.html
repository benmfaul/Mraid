MRAID stands for Mobile Rich media Ad Interface Definition and specifies an API for managing interactive creative within the mobile in-app environment. MRAID solves an analogous problem to VPAID, and was launched when the problem of managing interactive ads in a mobile environment became apparent. MRAID defines a general approach for HTML ads that display in any kind of container, but in practice applies to apps developed for mobile. The IAB currently supports two versions of MRAID: version 1.0 was released in 2011 while version 2.0 was released in 2012. Today, most native mobile apps implement one of these two versions.
